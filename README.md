## 🧭 Preparando o Ambiente

**Versão do Python3:** `python3.12` ou superior.

**Criar ambiente virtual do Python:**

Windows:

```sh
python -m venv .venv
```

Linux:

```sh
python3.12 -m venv .venv
```

**Instalar os requisitos:**

```sh
pip install -r requirements.txt
```
